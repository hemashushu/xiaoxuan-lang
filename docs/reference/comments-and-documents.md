注释和文档

行注释 //

结构注释（结构开关） #disable/#禁用

```js
#disable
if some then
  ...
else
  ...
end
```

块注释，用于描述一段代码的作用

```js
// 横线符号 `-` 或 `=` 的数量 >= 3
// 粗横线行上方或下方的连续行注释都属于范围注释的描述
// text etc.
// =============================
if some then {
   something
}

match some {
    case one:
        ...
    default:
        ...
}
// -----------------------------

// text etc.
// =============================
if some then {
   something
}
// -----------------------------
```

范围注释 /* .. */

文档注释，用在函数、结构体、字段等上方：
/// ...
/// ...

范围注释、块注释、文档注释都支持：
- markdown 富元素文档（图片：位图/矢量图svg/gif/apng/- webp、视频、音频等）
- 包含inline 测试
