# 集合

列表（List）

列表是一组数据类型相同的数据，其字面表示方法是：使用中括号包围所有数据，然后在每个数据之间使用逗号分隔。比如：

* `[1, 2, 3]`
* `[3.14, 2.718, 0.618]`
* `["天地玄黄", "宇宙洪荒", "日月盈昃", "辰宿列张"]`
* `[真, 假, 假, 真]`

一个列表里不能存在不同类型的数据，比如下列是错误的列表：

* `[100, 200, 2.9979e8, 6.67e-11]` // 前两个元素是整数型，后两个是浮点型。
* `["寒来暑往", 真, "秋收冬藏"]` // 前后两个是字符串型，中间是逻辑型。
